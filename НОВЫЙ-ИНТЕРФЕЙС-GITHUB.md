# Новый интерфейс GitHub: Как сделать Commit

GitHub изменил интерфейс! Теперь поле "Commit message" появляется **после** нажатия кнопки "Commit changes...".

---

## Как это работает сейчас:

### Шаг 1: Создай или отредактируй файл

1. Открой файл в редакторе (через "creating a new file" или редактирование существующего)
2. Введи имя файла вверху
3. Вставь код в текстовое поле

### Шаг 2: Найди кнопку "Commit changes..."

**В правом верхнем углу** страницы редактора есть зелёная кнопка:
- **"Commit changes..."** (если редактируешь существующий файл)
- **"Commit new file"** (если создаёшь новый файл)

### Шаг 3: Нажми кнопку

Нажми на зелёную кнопку **"Commit changes..."**

### Шаг 4: Появится окно с Commit message

После нажатия кнопки откроется окно/выпадающее меню с полями:

```
┌─────────────────────────────┐
│ Commit changes              │
│                             │
│ Commit message *            │
│ [Add requirements.txt]      │ ← СЮДА ПИШЕШЬ!
│                             │
│ [ ] Add an optional extended │
│     description...          │ ← Можно пропустить
│                             │
│ [Cancel] [Commit changes]   │ ← Нажми эту кнопку
└─────────────────────────────┘
```

### Шаг 5: Заполни Commit message

В поле **"Commit message"** введи описание, например:
- `Initial bot code`
- `Add requirements.txt`
- `Add Procfile for Railway`

### Шаг 6: Сохрани

Нажми зелёную кнопку **"Commit changes"** в этом окне.

```
│                                     │

│                                     │
│ ┌───────────────────────────────┐  │
│ │ Commit message *              │  │
│ │ [Initial bot code        ]    │  │ ← Шаг 5: Сюда пишешь
│ │                              │  │
│ │ [Cancel] [Commit changes]    │  │ ← Шаг 6: Нажми
│ └───────────────────────────────┘  │
└─────────────────────────────────────┘
```

---

## Важно:

- Поле "Commit message" **НЕ видно** на странице редактирования
- Оно появляется **только после** нажатия кнопки "Commit changes..."
- Это нормально для нового интерфейса GitHub!

---

## Если кнопка "Commit changes..." не работает:

1. Убедись, что ты внёс изменения в файл (добавил код)
2. Попробуй обновить страницу (F5)
3. Попробуй другой браузер (Chrome, Firefox, Edge)

---

## Готово!

После нажатия "Commit changes" файл сохранится в репозиторий, и ты сможешь перейти к следующему файлу.
