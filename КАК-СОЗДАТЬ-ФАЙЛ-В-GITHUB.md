# Как создать файл в GitHub (визуальная инструкция)

## Структура страницы редактора GitHub

Когда ты нажимаешь "creating a new file", открывается страница редактора. Вот как она выглядит:

```
┌─────────────────────────────────────────┐
│  [←] tg-wishlist-backend-bot /         │ ← Навигация
│                                         │
│  [main ▼] [bot.py]                     │ ← Имя файла (вверху)
│                                         │
│  ┌───────────────────────────────────┐ │
│  │                                   │ │
│  │  [Большое текстовое поле]        │ │ ← Сюда вставляешь код
│  │  для кода                         │ │
│  │                                   │ │
│  └───────────────────────────────────┘ │
│                                         │
│  ┌───────────────────────────────────┐ │
│  │ Commit new file                   │ │ ← Секция коммита
│  │                                   │ │
│  │ [ ] Add an optional extended      │ │ ← Опциональное описание
│  │     description...                │ │   (можно пропустить)
│  │                                   │ │
│  │ [Commit message *]                │ │ ← СЮДА ПИШЕШЬ!
│  │ [Initial bot code          ]     │ │   Обязательное поле
│  │                                   │ │
│  │ [Cancel] [Commit new file]        │ │ ← Кнопки
│  └───────────────────────────────────┘ │
└─────────────────────────────────────────┘
```

---

## Пошагово:

### Шаг 1: Открыть редактор

Нажми на синюю ссылку **"creating a new file"** на главной странице репозитория.

### Шаг 2: Ввести имя файла

**Вверху страницы** есть поле с текстом `[main ▼]` и пустое поле справа.

В это пустое поле введи имя файла, например: `bot.py`

### Шаг 3: Вставить код

**В большом текстовом поле ниже** (там может быть текст "Edit new file") вставь код файла.

### Шаг 4: Прокрутить вниз

**Прокрути страницу вниз** — там будет секция "Commit new file".

### Шаг 5: Заполнить Commit message

В секции "Commit new file" есть поле:

```
[Commit message *]
[                    ]
```

**СЮДА** введи текст, например: `Initial bot code`

Звёздочка (*) означает, что поле обязательное.

### Шаг 6: Нажать кнопку

Нажми зелёную кнопку **"Commit new file"** (внизу справа).

## Визуальный пример:

```
┌────────────────────────────────────┐
│ Имя файла (вверху):               │
│ [main ▼] [bot.py            ]     │ ← Шаг 2
│                                    │
│ ┌──────────────────────────────┐  │
│ │ import telebot              │  │ ← Шаг 3
│ │ ...                         │  │   (код файла)
│ └──────────────────────────────┘  │
│                                    │
│ [прокрути вниз ↓]                  │
│                                    │
│ ┌──────────────────────────────┐  │
│ │ Commit new file              │  │
│ │                              │  │
│ │ [ ] Add an optional...       │  │ ← Можно пропустить
│ │                              │  │
│ │ Commit message *             │  │
│ │ [Initial bot code      ]    │  │ ← Шаг 5: СЮДА!
│ │                              │  │
│ │ [Cancel] [Commit new file]  │  │ ← Шаг 6: Нажми эту
│ └──────────────────────────────┘  │
└────────────────────────────────────┘
```

---

## Важно:

- **Commit message** — это описание того, что ты делаешь
- Можно писать на русском или английском
- Это как комментарий к изменениям
- Поле обязательное (есть звёздочка *)

---

## Примеры Commit message:

- `Initial bot code`
- `Add bot.py`
- `Создаю файл бота`
- `Add requirements.txt`
- `Добавляю зависимости`

Можно писать что угодно, главное — описать что делаешь.
