# Пошаговая инструкция: Git + Netlify

## Часть 1: Регистрация на GitHub

1. Открой сайт **github.com** в браузере
2. Нажми **Sign up** (Зарегистрироваться)
3. Создай аккаунт: придумай логин, пароль, введи email
4. Подтверди email, если попросят

---

## Часть 2: Создать репозиторий на GitHub

1. Зайди на github.com (если ещё не зашёл)
2. Нажми зелёную кнопку **New** (или «New repository»)
3. Заполни:
   - **Repository name:** `tg-wishlist` (или любое имя)
   - **Public** должен быть выбран
   - **НЕ ставь галочку** «Add a README file»
4. Нажми **Create repository**
5. Скопируй URL репозитория (будет что-то вроде `https://github.com/ТВОЙ_ЛОГИН/tg-wishlist`)

---

## Часть 3: Установить Git (если ещё не установлен)

1. Открой **git-scm.com/download/win**
2. Скачай установщик для Windows
3. Запусти и нажимай **Next** везде (можно оставить настройки по умолчанию)
4. Дождись окончания установки

---

## Часть 4: Открыть терминал в папке проекта

1. Открой папку проекта в проводнике:  
   `C:\Users\tcndt\Documents\Cursor Projects\TG mini app`
2. В пустом месте папки **щёлкни правой кнопкой мыши**
3. Выбери **«Открыть в терминале»** или **«Git Bash Here»**
   - Если нет такой опции: нажми **Shift + правая кнопка** → «Открыть окно PowerShell здесь»

---

## Часть 5: Выполнить команды в терминале

Копируй команды **по одной**, вставляй в терминал (правой кнопкой или Ctrl+Shift+V) и нажимай **Enter**.

```bash
git init
```
(Напишет: «Initialized empty Git repository» — это хорошо)

```bash
git add .
```

```bash
git commit -m "Первый коммит"
```

```bash
git branch -M main
```

Теперь нужно добавить ссылку на твой репозиторий. **Замени** `ТВОЙ_ЛОГИН` и `tg-wishlist` на свои:
```bash
git remote add origin https://github.com/ТВОЙ_ЛОГИН/tg-wishlist.git
```

```bash
git push -u origin main
```
Тебя попросят войти в GitHub — введи логин и пароль (или авторизуйся через браузер).

---

## Часть 6: Подключить Netlify к GitHub

1. Зайди на **app.netlify.com**
2. Войди (если есть аккаунт) или зарегистрируйся через **Sign up**
3. Нажми **Add new site** → **Import an existing project**
4. Выбери **GitHub** (иконка с котиком)
5. Разреши доступ к репозиториям (нажми **Authorize Netlify**)
6. Найди свой репозиторий (`tg-wishlist`) и нажми **Import**
7. Настройки билда:
   - **Build command** — оставь пустым или удали то, что там есть
   - **Publish directory** — поставь `.` или `/`
8. Нажми **Deploy site**
9. Подожди 1–2 минуты — сайт будет задеплоен
10. Netlify даст тебе ссылку вида `https://название-сайта.netlify.app`

---

## Часть 7: Как обновлять сайт после изменений

Каждый раз, когда ты внёс изменения в проект:

1. Открой терминал в папке проекта (как в Части 4)
2. Выполни три команды:

```bash
git add .
```
```bash
git commit -m "Обновление"
```
```bash
git push
```

Через 1–2 минуты Netlify автоматически задеплоит новую версию.

---

## Если что-то пошло не так

- **Ошибка «git не найден»** — Git не установлен. Вернись к Части 3
- **Ошибка при git push** — проверь, что URL репозитория верный и ты вошёл в GitHub
- **Netlify не видит репозиторий** — убедись, что нажал «Authorize» при подключении GitHub
