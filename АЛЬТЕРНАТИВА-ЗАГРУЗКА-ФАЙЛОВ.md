# Альтернативный способ: Загрузить готовые файлы

Если не видишь поле "Commit message" при создании файла, используй загрузку готовых файлов.

---

## Способ 1: Загрузить файлы через "uploading an existing file"

### Шаг 1. Найди ссылку "uploading an existing file"

На главной странице пустого репозитория найди текст:

> **"Get started by creating a new file or uploading an existing file..."**

Нажми на синюю ссылку **"uploading an existing file"**.

### Шаг 2. Загрузи файлы

1. Откроется страница загрузки
2. Перетащи файлы из папки `bot-files` в окно браузера
   - Или нажми **"choose your files"** и выбери файлы
3. Загрузи файлы:
   - `bot.py`
   - `requirements.txt`
   - `Procfile`
   - `README.md` (опционально)

### Шаг 3. Commit

После загрузки файлов внизу страницы будет кнопка **"Commit changes"** — нажми её.

---

## Способ 2: Использовать готовые файлы из этого проекта

Файлы уже готовы в папке `bot-files` этого проекта. Можешь:

1. Открыть папку `bot-files` на компьютере
2. Скопировать содержимое файлов
3. Создать файлы в GitHub через "creating a new file"
4. Вставить содержимое

---

## Способ 3: Если интерфейс GitHub другой

Если интерфейс GitHub выглядит по-другому:

1. Попробуй обновить страницу (F5)
2. Или используй прямую ссылку для создания файла:
   ```
   https://github.com/ТВОЙ_ЛОГИН/tg-wishlist-backend-bot/new/main
   ```
   (замени ТВОЙ_ЛОГИН на свой логин)

3. После ввода кода и имени файла — прокрути страницу в самый низ
4. Там должна быть кнопка или поле для коммита

---

## Способ 4: Через GitHub Desktop (если установлен)

Если у тебя установлен GitHub Desktop:

1. Открой GitHub Desktop
2. Клонируй репозиторий `tg-wishlist-backend-bot`
3. Скопируй файлы из `bot-files` в папку репозитория
4. В GitHub Desktop нажми **Commit** → **Push**

---

## Самый простой способ (рекомендую):

1. Открой папку `bot-files` в этом проекте
2. Открой каждый файл (bot.py, requirements.txt, Procfile)
3. Скопируй содержимое
4. В GitHub:
   - Нажми "creating a new file"
   - Введи имя файла
   - Вставь содержимое
   - **Прокрути страницу в самый низ** — там должна быть кнопка сохранения

Если кнопки всё равно нет — сделай скриншот всей страницы (включая низ), и я подскажу точнее!
