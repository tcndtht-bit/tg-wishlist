# Где найти "Add file" в пустом репозитории GitHub

В пустом (новом) репозитории GitHub кнопка "Add file" выглядит по-другому.

---

## Способ 1: Через ссылку "creating a new file" (в пустом репозитории)

Если репозиторий пустой, на главной странице есть текст:

> **"Get started by creating a new file or uploading an existing file..."**

1. Найди эту надпись на странице (она в центре, под URL репозитория)
2. Нажми на синюю ссылку **"creating a new file"** (или **"uploading an existing file"**)
3. Откроется редактор для создания файла

---

## Способ 2: Через кнопку "Add file" (если репозиторий не пустой)

Если в репозитории уже есть файлы:

1. На главной странице репозитория (вкладка **Code**)
2. Справа от кнопки **Code** (зелёная) есть кнопка **"Add file"**
3. Нажми на неё → выбери **"Create new file"** или **"Upload files"**

---

## Способ 3: Через URL напрямую

Можешь открыть редактор напрямую по ссылке:

```
https://github.com/ТВОЙ_ЛОГИН/tg-wishlist-backend-bot/new/main
```

Замени:
- `ТВОЙ_ЛОГИН` на свой логин GitHub
- `tg-wishlist-backend-bot` на имя репозитория

---

## Пошаговая инструкция для пустого репозитория

### Шаг 1: Найди секцию "Quick setup"

На главной странице репозитория найди текст:

```
Quick setup — if you've done this kind of thing before
```

### Шаг 2: Под этой секцией найди текст

```
Get started by creating a new file or uploading an existing file.
We recommend every repository include a README, LICENSE, and .gitignore.
```

### Шаг 3: Нажми на "creating a new file"

Слово **"creating a new file"** — это синяя ссылка. Нажми на неё.

### Шаг 4: Откроется редактор

Теперь ты увидишь:
- Поле для имени файла вверху
- Большое текстовое поле для кода
- Кнопку "Commit new file" внизу

---

## Если всё равно не видишь

### Вариант A: Добавь README через GitHub

1. Открой репозиторий
2. В правом верхнем углу нажми **Settings**
3. Прокрути вниз до секции **"Features"**
4. Поставь галочку **"Initialize this repository with a README"**
5. Нажми **Update**
6. После этого появится кнопка **"Add file"** на главной странице

### Вариант B: Используй веб-интерфейс GitHub

1. Открой репозиторий
2. В адресной строке браузера допиши: `/new/main`
3. Например: `https://github.com/tcndtht-bit/tg-wishlist-backend-bot/new/main`
4. Откроется редактор

## Альтернатива: Загрузить файлы через веб-интерфейс

Если не можешь найти "Add file", можешь загрузить готовые файлы:

1. Найди текст **"uploading an existing file"** (синяя ссылка)
2. Нажми на неё
3. Перетащи файлы из папки `bot-files` (из этого проекта) в окно браузера
4. Или нажми **"choose your files"** и выбери файлы

---

## Готовые файлы для загрузки

Файлы уже готовы в папке `bot-files` этого проекта:
- `bot.py`
- `requirements.txt`
- `Procfile`
- `README.md`

Просто загрузи их в репозиторий через "uploading an existing file".
