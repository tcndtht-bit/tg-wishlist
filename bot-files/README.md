# Telegram Wishlist Bot

Бот для создания карточек желаний из фотографий товаров.

## Переменные окружения

- `BOT_TOKEN` - токен бота из BotFather
- `WEB_APP_URL` - URL мини-приложения (опционально)

## Деплой на Railway

1. Подключи этот репозиторий к Railway
2. Добавь переменные окружения `BOT_TOKEN` и `WEB_APP_URL`
3. Railway автоматически задеплоит бота
